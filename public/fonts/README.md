# Fonts Directory

Place the Andellia Davilton font file here:

- File name: `AndelliaDavilton.otf`
- Location: `/public/fonts/AndelliaDavilton.otf`

You can download the font from:
- https://www.dafont.com/andellia-davilton.font (free for personal use)
- https://hanzelspace.com/product/andellia-davilton/ (commercial license)

After placing the font file, restart your development server for the changes to take effect.
